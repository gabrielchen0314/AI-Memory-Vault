# 模板系統

> 建立新內容時，請先查閱對應模板。

---

## 模板分類

### Agent 角色模板 — `agents/`

定義每個 Agent 的職責和輸出格式。

### 專案類型模板 — `projects/`

| 類型 | 用途 |
|------|------|
| `python-app/` | Python 應用程式 |
| `unity-game/` | Unity 遊戲專案 |
| `vscode-ext/` | VS Code 擴充套件 |

### 區域結構模板 — `sections/`

| 區域 | 用途 |
|------|------|
| `company-workspace/` | 公司工作域 |
| `rules/` | 編碼規範 |
| `meeting/` | 會議紀錄 |
| `journal/` | 日記 |
| `learning/` | 學習筆記 |
| `knowledge-card/` | 知識卡片 |
