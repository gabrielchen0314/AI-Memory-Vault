## 模板分類

### Agent 角色模板 — `agents/`

定義每個 Agent 的職責和輸出格式。

| 檔案 | Agent | 說明 |
|------|-------|------|
| `architect.md` | `@Architect` | 系統架構設計 |
| `build-error-resolver.md` | `@BuildErrorResolver` | 建置錯誤解決 |
| `code-reviewer.md` | `@CodeReviewer` | 程式碼審查 |
| `doc-updater.md` | `@DocUpdater` | 文件與 API Map 更新 |
| `git-committer.md` | `@GitCommitter` | Git 提交流程 |
| `learn-trigger.md` | `@LearnTrigger` | 學習觸發與 Instinct 建立 |
| `planner.md` | `@Planner` | 任務規劃分解 |
| `refactor-cleaner.md` | `@RefactorCleaner` | 重構與死碼清理 |
| `retrospective.md` | `@Retrospective` | 月度復盤分析 |
| `rm-issue-creator.md` | `@RmIssueCreator` | Redmine 開單 |
| `rm-tracker.md` | `@RmTracker` | RM 追蹤分析 |
| `security-reviewer.md` | `@SecurityReviewer` | 遊戲安全審查 |
| `skill-author.md` | `@SkillAuthor` | Agent Skill 撰寫 |
| `tdd-guide.md` | `@TddGuide` | TDD 開發導引 |
